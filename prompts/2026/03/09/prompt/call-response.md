I just copied the ARC Journal project to a new Figma Make project named "Call + Response". This is intended to be an extension of the ARC system for logging Figma Make prompts and responses to build a connection between the prompt in plain text, the response in Markdown, and the GitHub repository commits. Then, we can build a complete system for understanding the state of multiple projects with a build log, timeline, and milestones.
